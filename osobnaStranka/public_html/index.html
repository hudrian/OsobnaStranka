Something about me

I always like computers, programming and communication with external devices. Already on the middle school I wrote several programs to control external robotic engines and self made printers. Starting with serial and parallel port interfaces, later building my own slot cards as interface. Made an interface for disco light controlling system – including high voltage optical switch. Went to university to study Computer Science. My first job was designing and programming control system for checking car parts finalization. Specially communicating with scanners and cameras, matching different parts bar codes and making logic for the system. - This was done for BMW part supplier (RF). Currently working as a software engineer at Mühlbauer on mailing system machines , designing and programming the system, finding ways to optimize process for speed, accuracy and stability. I'm personally one of the main persons working on our latest mailing machine with completely redesigned mechanics and software. I like challenging work, working with latest technologies, willing to learn, have lot of debugging and optimization skills.
